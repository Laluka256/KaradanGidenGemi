# KaradanGidenGemi
bu projemiz side-Scroll aksiyon oyunu olacaktır ve karadan gemiyi yürütemediğimiz için yürüyerek düşmanlarımızı avlamaya koyulduğumuz bir yeniçeri olarak oynayacağız.

 Ekibimizdeki kişiler: Oktay Kaya, Yiğit Topsan, Berat Konca.

projemizde AntiGravity ve Gemini kullanıyoruz.
